如何使用這個軟體？ | How to use this software?
  下載原始程式碼tarball（或是把原始程式碼倉庫整個 clone 下來），以系統管理權限執行 Installer.sh script程式就會把軟體安裝在 /usr/local/share/Lazy_Youtube_Downloader 並建立 symbolic式連結「lazy-youtube-dl」至 /usr/local/bin 
  執行「lazy-youtube-dl」命令，依照提示操作
  
本軟體的軟體依賴特性 | Software dependency of this software
  本軟體依賴「youtube-dl」命令才能正常運作
  ubuntu散佈版中的軟體包裹名稱為「youtube-dl」

本軟體的用途 | Usage of this software
  用來把好看的youtube影片下載到當前工作目錄(current working directory)，對於使用 Dolphin檔案管理軟體的人來說這樣相當方便
